# 方寸 (Square Inch) - 个性化网站导航

> 于方寸之间，见天地之宽

一个简洁优雅的 Chrome 浏览器扩展，提供个性化的网站导航体验。支持任意协议URL、智能标签管理、内联删除确认等现代化功能。

## ✨ 核心特性

- 🎯 **简洁设计** - 现代化卡片式布局，优雅美观
- ⚡ **快速访问** - 一键访问常用网站，支持所有协议（http、https、chrome://、ftp等）
- 🏷️ **智能标签** - 分页加载标签管理，滚动查看更多，固定新增按钮
- 🔍 **智能搜索** - 支持网站名称和网址的模糊搜索
- 🗑️ **内联删除** - 直接在卡片内确认删除，带滑入动画，全局互斥
- 🎨 **深色模式** - 支持明暗主题切换，护眼舒适
- 📱 **响应式设计** - 完美适配桌面、平板、手机
- 💾 **本地存储** - 数据安全存储在本地，隐私保护
- 📤 **导入导出** - 支持数据备份和跨设备同步
- 🖱️ **右键快捷** - 任意页面右键快速添加到导航

## 🚀 安装

### 开发者模式安装
1. 下载或克隆此仓库到本地
2. 打开 Chrome 浏览器，进入 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目文件夹

## 🔒 权限说明

方寸扩展仅使用以下3个必要权限：

### **activeTab（活动标签权限）**
**方寸使用activeTab权限是为了**获取您当前浏览网页的网址信息。当您在任意网页上右键选择"添加到方寸导航"时，扩展需要读取当前页面的URL（如 https://example.com），以便自动填充到添加网站的表单中，让您无需手动输入网址。

### **contextMenus（右键菜单权限）**
**方寸使用contextMenus权限是为了**在网页右键菜单中添加快捷操作选项。这样您可以在任意网页上右键看到"打开方寸导航"和"添加到方寸导航"两个选项，提供更便捷的使用体验，无需每次都点击浏览器工具栏的扩展图标。

### **tabs（选项卡权限）**
**方寸使用tabs权限是为了**在新标签页中打开导航页面和支持特殊网址访问。当您点击右键菜单中的"打开方寸导航"时，扩展会在新标签页中打开导航页面；同时支持您添加和访问chrome://等浏览器内部页面（如设置页、扩展管理页等特殊协议网址）。

**🔒 隐私保护承诺**：
- 方寸的所有数据仅存储在您的本地浏览器中
- 方寸不会上传、分享或泄露任何个人信息
- 方寸不会追踪您的浏览记录和行为
- 方寸只在您主动操作时才访问当前页面信息

## 📖 使用指南

### 🌐 网站管理
- **添加网站**: 点击 "+" 卡片或右键菜单"添加到方寸导航"
- **编辑网站**: 悬停网站卡片，点击编辑按钮
- **删除网站**: 悬停网站卡片，点击删除按钮，卡片内确认删除
- **访问网站**: 直接点击网站卡片跳转
- **协议支持**: HTTP、HTTPS、Chrome内部协议(chrome://)、FTP、自定义协议等

### 🏷️ 标签功能
- **查看标签**: 左侧显示所有标签，点击筛选网站
- **添加标签**: 点击左侧底部 "+" 按钮或标签选择器内新增
- **删除标签**: 悬停标签项，点击删除按钮，横向确认删除
- **智能选择**: 标签选择器显示4.5个标签+底部固定新增按钮，滚动加载更多

### 🔍 搜索功能
- 实时搜索：在搜索框输入关键词进行模糊匹配
- 支持搜索：网站名称、网址、域名
- 清除搜索：点击清除按钮或删除所有文字

### 🎨 主题切换
- 点击右上角月亮/太阳图标切换深色/明亮模式
- 主题偏好自动保存，下次打开恢复

### 📤 数据管理
- **导出数据**: 点击导出按钮下载 JSON 格式备份文件
- **导入数据**: 点击导入按钮选择备份文件
- **智能导入**: 自动判断新增或覆盖，支持选择性导入

### 🖱️ 快捷访问
- **右键菜单**: 任意页面右键选择"方寸导航"选项
- **快速添加**: 右键选择"添加到方寸导航"自动填充当前网站
- **弹窗访问**: 点击浏览器工具栏插件图标

## 🛠️ 技术栈

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **框架**: Chrome Extension Manifest V3
- **存储**: localStorage（本地存储）
- **权限**: activeTab, contextMenus, tabs
- **样式**: CSS Grid + Flexbox 响应式布局
